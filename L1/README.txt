空間智慧互動式 HTML 投影片 v4-fixed

這版修正重點：
- index.html 是「可直接開啟」的完整互動投影片，內容、CSS、JS 都已內嵌。
- assets/runtime.js 只是備份，不是必要依賴；就算只有 index.html 也應該能看到完整內容。
- 請不要在 Google Drive 預覽模式判斷 HTML 是否正常；Drive 不適合直接預覽互動網頁。請下載後用 Chrome / Safari 開啟 index.html。

使用方式：
1. 下載整個 ZIP 或 Drive 資料夾。
2. 解壓縮。
3. 直接點 index.html。

操作：
- ← / → / Space：換頁
- Home / End：第一頁 / 最後一頁
- O：目錄
- S：演講者模式
- F：全螢幕
- #/1、#/2：指定頁碼
