一小時 Python 入門 Part 1

把網頁文章改編成適合教學與分享的 HTML 投影片：從 Hello World 到 List。

改編來源：寫點科普 Lynn〈一小時Python入門-part 1〉

Why Python

為什麼初學者適合 Python？

簡潔易讀

語法接近自然語言，初學者能更快把注意力放在「邏輯」而不是符號。

用途很廣

資料分析、自動化、AI、網站、爬蟲與商業流程都能用。

主流入門語言

許多大學與 CS 課程都把 Python 作為第一門程式語言。

Setup

環境設置：先能執行，再開始學

初學最重要不是裝很多工具，而是先確定：你能寫、能跑、能看到結果。

兩種常見方式

① 安裝 Python + 用 Terminal 執行 `.py` 檔

② 使用 Jupyter Notebook 互動式練習

教學場景建議用 Notebook，回饋快、比較不挫折。

01 Print

第一個程式：跟世界打招呼

`print()` 會把你放進括號裡的內容印出來。

# Hello World
print("Hello, Python!")

name = "Jason"
print(name)
print(1234)
02 Variable

變數：幫資料取一個名字

整數 int

`a = 123`
適合計數、年齡、數量。

字串 str

`b = "456"`
看起來像數字，但本質是文字。

浮點數 float

`c = 8.70`
適合小數、比例、分數。

很多 TypeError 都來自：把不同型態的資料硬湊在一起。

Type Conversion

型態轉換：讓資料變成你要的樣子

輸入進來的資料常常是字串；要計算前，記得轉成數字。

x = "456"
y = int(x)
print(y + 10)   # 466

age = 18
print("我今年 " + str(age) + " 歲")
03 Input

input：讓使用者把資料交給程式

`input()` 會等待使用者輸入，而且收到的一律先是字串。

name = input("Your name: ")
print("Hello, " + name)

n = int(input("請輸入數字: "))
print(n * 2)
Practice 1

小練習：四則運算

任務

讀取兩個整數，印出加、減、乘、除結果。

提示：`input()` 得到的是字串，要先 `int()`。

a = int(input("a = "))
b = int(input("b = "))
print(a + b)
print(a - b)
print(a * b)
print(a / b)
04 If / Else

條件判斷：讓程式做選擇

每天都在 if / else：如果下雨就帶傘，否則就直接出門。

battery = 50

if battery >= 80:
    print("去抓寶可夢")
elif battery >= 30:
    print("看看小說就好")
else:
    print("回家充電")
Logic

常用邏輯符號

> 大於< 小於>= 大於等於== 等於!= 不等於and 且or 或not 非True / False

注意：判斷相等是 `==`，不是指定變數用的 `=`。

05 For Loop

迴圈：讓重複的事自動發生

不要手寫 100 行 `print()`。交給迴圈，程式才像程式。

# 印出 0 到 9
for i in range(10):
    print(i)

# range(起點, 終點, 間距)
for i in range(1, 10, 2):
    print(i)
Nested Loop

小練習：九九乘法表

重點

for loop 裡面可以再放一個 for loop。

外層控制第幾列，內層控制每列要印什麼。

for i in range(1, 10):
    for j in range(1, 10):
        print(i * j, end="\t")
    print()
06 List

List：能裝很多資料的格子

當資料不只一筆，就不要一直開 `a、b、c`；用 List 管理一串資料。

a = ["Lynn", 0.87, 1234, True]

print(a[0])   # Lynn
print(a[3])   # True
print(len(a))  # 4
List Tools

List 常用操作

新增

`append(x)` 放到最後
`insert(i, x)` 插到指定位置

刪除

`pop()` 刪最後
`remove(x)` 刪第一個 x
`clear()` 清空

計算

`max()` 最大值
`min()` 最小值
`sum()` 總和

切片

`list[start:end]`
取出 start 到 end-1 的部分。

Practice 2

成績改造：List + Loop

把五次成績開根號再乘以 10，算新平均。

scores = [10, 30, 50, 70, 90]
new_scores = []

for s in scores:
    new_scores.append(s ** 0.5 * 10)

print(sum(scores) / len(scores))
print(sum(new_scores) / len(new_scores))
