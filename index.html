🧠 單元1：什麼是 LLM？

大型語言模型（LLM, Large Language Model）是一種透過大量文本資料訓練的 AI 模型。它學會了語言的規律，能夠理解並生成自然語言。

核心概念：Token 預測
LLM 的本質是「根據前面的文字，預測下一個最可能出現的 Token」。透過數十億個參數，它學會了語言的統計規律。

常見的 LLM 包括：OpenAI 的 GPT 系列、Anthropic 的 Claude、Google 的 Gemini。

🤖 單元2：什麼是 AI Agent？

AI Agent 比一般的問答 AI 更進一步。它能夠：

  • 規劃：把複雜目標拆解成多個步驟
  • 使用工具：呼叫搜尋、計算、寫程式等外部工具
  • 記憶：記住對話歷史和重要資訊
  • 自主行動：不需要人類每一步指令
ReAct 框架
Reason（推理）→ Act（行動）→ Observe（觀察）→ 反覆循環，直到完成目標

💻 單元3：第一個 Claude API 程式

以下是用 Python 呼叫 Claude API 的基本範例：

# 安裝套件 # pip install anthropic import anthropic # 建立客戶端 client = anthropic.Anthropic( api_key="your-api-key-here" ) # 發送訊息 message = client.messages.create( model="claude-opus-4-6", max_tokens=1024, messages=[ {"role": "user", "content": "什麼是 AI Agent？"} ] ) print(message.content[0].text)

🛠 單元4：Prompt Engineering

好的 Prompt 包含三個要素：

  • 角色：告訴 AI 它是誰（System Prompt）
  • 任務：清楚說明要做什麼
  • 格式：指定輸出的格式
# 帶有 System Prompt 的範例 message = client.messages.create( model="claude-opus-4-6", max_tokens=1024, system="你是一位專業的 Python 老師，用繁體中文回答，解釋要簡單易懂。", messages=[ {"role": "user", "content": "解釋什麼是 for 迴圈，給初學者聽"} ] )
