AI Consultant View

如果導入 AI，毛天使最適合先做 5 個 Skills

1. 情境導購 Skill

先問毛孩種類、年齡、外出頻率、居家味道，再推薦組合。

2. 商品知識庫 Skill

整理成分、用途、使用方式、注意事項與搭配產品。

3. 寵物知識 FAQ Skill

把部落格文章轉成客服問答與教育型導購。

4. 回購提醒 Skill

針對噴霧、補充瓶、洗毛精估算使用週期並提醒補貨。

5. 社群內容 Skill

把情境、知識文、顧客回饋轉成短影音腳本與貼文。
